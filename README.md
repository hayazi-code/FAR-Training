# FAR Explorer

A React + Vite + TypeScript educational zoning and massing tool with a large 3D workspace, collapsible controls sidebar, multilingual UI, and shareable feet-based parcel settings.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## GitHub Pages

This repo includes a GitHub Actions workflow at `.github/workflows/deploy-pages.yml` that builds and deploys the app to GitHub Pages whenever `main` is updated.
